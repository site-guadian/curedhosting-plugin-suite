Stripe Payment Module
=====================

This package contains a standalone WordPress plugin that is ready to be wired up for Stripe Checkout.

What is included
----------------
- A simple admin settings page for Stripe configuration
- Quick checkout buttons for Pro and Corporate plans
- A webhook endpoint for Stripe events
- Ready-to-fill fields for Stripe keys, price IDs, and URLs

What you need later
-------------------
Add these values in the plugin settings page:
- Stripe secret key
- Stripe publishable key
- Stripe webhook secret
- Pro price ID
- Corporate price ID
- Success URL
- Cancel URL

How to use it
-------------
1. Upload the plugin folder to your WordPress site.
2. Activate Stripe Payment Module from the Plugins screen.
3. Open Stripe Payments in the WordPress admin.
4. Fill in your Stripe keys and price IDs.
5. Add the webhook URL shown on the page to your Stripe dashboard.
6. Use the quick checkout buttons to test the flow.
